# Olivetti Prodest PC1 - Complete 512 Color Reference

The V6355D video chip provides 512 colors using 3-bit RGB (0-7 per channel).

## Color Formula
- **Color Index** = R×64 + G×8 + B
- **DAC Byte 1** = R (0-7)
- **DAC Byte 2** = (G << 4) | B
- **6-bit value** = 3-bit × 9 (for PC1PAL files)
